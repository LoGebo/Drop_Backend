API de Seguimiento de Transporte en Tiempo Real

Esta página proporciona información sobre los endpoints disponibles en la API.

La documentación completa está disponible en: /api-docs

Endpoints de Vehículos

GET /api/vehicles

Obtener todos los vehículos disponibles.

GET /api/vehicles/:vehicleId

Obtener información de un vehículo específico.

GET /api/vehicles/type/:vehicleType

Obtener vehículos por tipo (BUS, TRAIN, etc.).

GET /api/vehicles/route/:routeId

Obtener vehículos que están en una ruta específica.

POST /api/vehicles/:vehicleId/location

Actualizar la ubicación de un vehículo.

{
  "latitude": 19.432608,
  "longitude": -99.133209,
  "occupancy": 45,
  "route": "ROUTE1"
}

Endpoints de Rutas

GET /api/routes

Obtener todas las rutas disponibles.

GET /api/routes/:routeId

Obtener información de una ruta específica.

GET /api/routes/:routeId/vehicles

Obtener todos los vehículos que están en una ruta específica.

POST /api/routes

Crear una nueva ruta.

{
  "id": "ROUTE3",
  "name": "Ruta Centro-Norte",
  "type": "BUS",
  "stops": [
    {
      "id": "STOP1",
      "name": "Estación Central",
      "coordinates": {
        "latitude": 19.432608,
        "longitude": -99.133209
      }
    },
    {
      "id": "STOP8",
      "name": "Terminal Norte",
      "coordinates": {
        "latitude": 19.452519,
        "longitude": -99.146320
      }
    }
  ]
}

POST /api/routes/suggest

Obtener sugerencia de ruta óptima entre dos puntos.

{
  "origin": {
    "latitude": 19.432608,
    "longitude": -99.133209
  },
  "destination": {
    "latitude": 19.414562,
    "longitude": -99.167721
  }
}
